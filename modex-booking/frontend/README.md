# Frontend - Doctor Appointment System

## Setup Instructions

### Prerequisites
- Node.js (v14+)

### Installation
1.  Navigate to `frontend` directory.
2.  Run `npm install`.

### Configuration
1.  Create `.env` file in `frontend` directory.
2.  Add `REACT_APP_API_URL=http://localhost:5000/api` (or your backend URL).

### Running the App
- Development: `npm start`
- Production Build: `npm run build`

## Features
-   **User**: Register, Login, View Shows, Book Appointment, View/Cancel My Bookings.
-   **Admin**: Create Shows, Manage Shows, View Bookings (Placeholder).
-   **Security**: JWT Authentication, Protected Routes.

## Tech Stack
-   React (TypeScript)
-   Tailwind CSS
-   Axios
-   React Router DOM
